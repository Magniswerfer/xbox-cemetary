# Xbox Cemetery

## Design Context

This project is set up with impeccable design context. Read these before design work:

- **[PRODUCT.md](PRODUCT.md)** — register, users, brand personality, anti-references, design principles.
- **[DESIGN.md](DESIGN.md)** — visual system: colors, typography, elevation, components, do's/don'ts.
- **`.impeccable/design.json`** — machine-readable token + component sidecar (powers `/impeccable live`).

**Register:** brand (a memorial experience; design IS the product).

**Audience:** gaming-history fans / preservation-minded players. Goal — make studio closures feel real and specific, with the facts intact.

**Voice:** elegiac, nostalgic, wry. A eulogist with dry, bittersweet humor. A wake, not a roast.

**North Star:** "The Jewel-Case Reliquary" — the translucent green plastic shell of an original Xbox, hollowed into a reliquary for the dead.

**Do NOT make it look like:** the official Xbox/Microsoft site · generic gamer RGB dark UI · a sterile wiki/data table · jokey meme content.

**Accessibility bar:** WCAG 2.1 AA + honor `prefers-reduced-motion`; the mood must survive with effects dialed back.
